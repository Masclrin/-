# 宏执行框架快速使用说明

仅针对 `宏执行框架*.py` 系列。
当前建议直接使用：`宏执行框架2.1.py`。

## 1. 运行要求

1. Windows 10/11
2. Python 3.10+
3. 管理员权限（安装/卸载 Interception 驱动时需要）

## 2. 首次安装

推荐顺序（最省事）：

1. 双击 `1_安装依赖.bat`
2. 右键管理员运行 `2_安装驱动_管理员运行.bat`
3. 重启电脑

手动安装命令：

```powershell
pip install -r requirements.txt
cd "Interception\command line installer"
.\install-interception.exe /install
```

## 3. 启动脚本

宏执行需要管理员权限，请确保以管理员执行！

```powershell
python "宏执行框架2.1.py"
```

或执行：`3_启动宏.bat`

看到以下日志之一即可：

1. `Interception 驱动已预加载`：内核注入模式（精度更高）
2. `Interception 不可用，回退 SendInput`：兼容模式（可运行）

## 4. 常用操作

1. 修改触发与宏映射：`MACRO_BINDINGS`
2. 修改运行行为：`MACRO_RUNTIME_SETTINGS`
3. 急停全部宏：`ESC`
4. 全局暂停触发（2.0+）：`F8`

2.1 默认支持：

1. `running_press_mode = hold_pause`：按住触发键暂停，抬起继续
2. 前馈补偿默认启用（`LAG_USE_FEEDFORWARD = True`）

## 5. 版本改动速览（1.8 -> 2.1）

1. `1.8`：建立基础框架完善（JSON 宏高精度执行、PI误差补偿、Interception/SendInput 双后端）
2. `1.9`：修复左键无法触发的问题，完善并发执行管理
3. `1.91`：加入 REALTIME / Pro Audio 激进调度实验
4. `1.92`：调度能力改为可开关，PI 参数回调至更稳健区间
5. `1.93_alpha`：接入 v5b 引擎与智能控制器路线实验
6. `1.93_beta自适应PI`：加入自适应增益、振荡检测、软启动
7. `1.94`：升级 PID（含微分），增强限幅与防震荡机制。用于长宏
8. `2.0`：引入运行态交互控制（F8 全局暂停、按键二次行为、抬起暂停），并加入前馈控制路线
9. `2.1`：新增 `hold_pause`，默认切到前馈补偿，优化默认输出策略

## 6. 快速排错

1. 外部键鼠干涉失效：检查 `interception_input.py` 中 `INTERCEPTION_CAPTURE_PHYSICAL_INPUT = False`
2. 无法创建 Interception 上下文：确认驱动已管理员安装并重启
3. 脚本启动但无动作：检查 `MACRO_BINDINGS` 对应宏文件路径是否正确

## 7. 卸载驱动（可选）

```powershell
cd "Interception\command line installer"
.\install-interception.exe /uninstall
```
